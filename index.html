Курс по урегулированию задолженности

Информация

Этот курс предназначен для людей, которые хотят научиться эффективно работать с задолженностями, помогать клиентам решать финансовые проблемы и зарабатывать на этом. Вы получите практичные знания и инструменты для работы с организациями (МФО), банками второго уровня, а также по снятию арестов.

Пакеты

Базовый пакет

  • - Обучение по работе с микрофинансовыми организациями (МФО).
  • - Пакет документов для взаимодействия с МФО.
  • - Снятие арестов: пошаговая инструкция.
  • - 2 недели теоретического обучения.
  • - 1 месяц чата поддержки и консультаций.
